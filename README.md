# JustPDF

A simple, fast, and intuitive PDF viewer built with Flutter, now available on the Google Play Store.

## Overview

**JustPDF** is a lightweight PDF viewer designed for quick and easy access to your documents. With essential features like adding favorites, sharing, and printing, JustPDF streamlines document management for everyday users. Ideal for anyone looking for a straightforward and efficient way to view and organize PDFs.

## Features

- **Easy PDF Viewing**: Open and navigate PDF files smoothly on any device.
- **Favorites**: Mark frequently used files as favorites for quick access.
- **Share**: Share PDFs directly from the app.
- **Print**: Print documents directly via compatible printers.
- **Minimalist Interface**: Enjoy a clutter-free and user-friendly interface.

## Getting Started

This project is built using Flutter. To set up and run JustPDF locally, make sure you have Flutter installed.

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart](https://dart.dev/get-dart)

### Code Generation

To generate required code, use the following command:

    ```bash
    flutter pub run build_runner build --delete-conflicting-outputs
    ```

### Running Tests

Ensure all necessary code is generated, then use this command to run tests:

    ```bash
    flutter test
    ```

If you prefer to view test previews on a device or emulator, run:

    ```bash
    flutter run path/to/your/test.dart
    ```

While running, type R in the console for a hot restart, which will show test results on the screen.

## Installation

### From Source

1. Clone this repository:

    ```bash
    git clone https://github.com/AntoniPochopien/just_pdf.git
    ```

2. Navigate to the project directory:

    ```bash
    cd justpdf
    ```

3. Install dependencies:

    ```bash
    flutter pub get
    ```

4. Run code generator:

    ```bash
    flutter pub run build_runner build --delete-conflicting-outputs
    ```

5. Run the app:

    ```bash
    flutter run
    ```

From Google Play Store
JustPDF is available for download on the Google Play Store [**Here**](https://play.google.com/store/apps/details?id=pl.byteitright.justpdf).
